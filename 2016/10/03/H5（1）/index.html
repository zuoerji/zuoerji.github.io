    智能表单

    • 新的表单类型，不建议在网页中使用
    • 可以针对需求使用特定的只能表单类型
      • <form action="#" method=""></form>
      • 之间学过的表单标签：file text password radio checkbox image button submit button reset select
      • button元素和type=“button”
      • <input type="datetime">兼容性不佳，可以用<input type="date">不能在PC上用，样式无法自定义
      • <input type="color">在移动端不支持
      • <input type="range" min="1" max="10" step="5" value="5"> 定义带有 slider 控件的数字字段。value：默认 step：单步选择
      • <input type="email" value="some@qq.com" pattern="正则表达式">本身具备校验
      • <input type="text" required>必填选项，常用于移动端，无法设置样式
      • <input type="tel" placeholder="(555)555-5555" pattern="">本身不具备校验
      • input：range + tab 快捷键

    SVG

    • Scalable Vector Graphics：可缩放矢量图形

    • 位图就是由一堆像素点组成，由有颜色的像素

    • 矢量图记录的是线，路径，填充色，任意缩放，图片不会失真，基于可扩展标记语言

    • 文档标题<?xml version=”1.0” encoding=”utf-8”……> 为xml文档

    • 优点：体积小（数据量少），质量高（放大无锯齿），效果好，可控程度高

    • 定义 线性渐变 stop circle 定义了一个圆 定义了url(#id名)

    • 可以用CSS填充颜色

    • 图标字体

    • 以文件的方式使用 推荐使用，不能通过CSS控制样式

    • 在标准中有定义

    • 在标准中未定义，但浏览器都支持

    • 浏览器不支持，学了异步之后
