    一、正则表达式

    • regular expression——一个有规则的表达式
    • 简单的说 他就是一个用于 查找的通配符
    • 正则表达式就是 一个用于查找的 含有 匹配字符串 或 匹配元字符的 字符串
    • 正则表达式对象
      • js 正则表达式对象 就是 由 正则表达式创建的对象, 该对象可以进行 匹配, 提取 和 替换.
      • 创建正则表达式对象
        • 构造函数
          • var reg = new RegExp( 正则表达式字符串[, 匹配模式] );
          • 字面量
          • var regex = /正则表达式/;
    • 使用正则表达式进行匹配
      • 语法:
      • 正则对象.test( 字符串 ) -> bool 如果参数字符串中含有 复合 正则匹配的 子字符串, 就返回 true, 否则返回 false

    二、基本元字符

    • . 表示任一个非换行的字符
    • () 表示分组和提高优先级
    • [] 表示一个字符, 出现在 [] 中的字符
      用法: [abc] 出现 [] 中的任意一个字符
    • | 或 用法: 正则表达式|正则表达式
    • 扩展
      • . 任意的一个字符. 没有任何限制
      • [] 是出现在[]中的一个字符. 认为 . 的限制级版本
      • | 可以认为是 允许使用多个字符的 匹配
    • 转义字符: \
      • 表示点: .
      • 表示[]: [ ]
      • 表示(): ( )
      • 表示\ : \

    三、限定元字符

    • ‘*’ 紧跟前面的一个字符或一组字符出现 0 次到多次
      • 紧跟在前面的字符出现 1 次到多次
    • ? 紧跟在前面的字符出现 0 次或 1 次
    • {数字} 紧跟在前面的字符出现指定次数
    • {数字,} 紧跟在前面的字符至少出现指定次数
    • {数字, 数字} 紧跟在前面的字符出现的次数范围

    四、首尾正则表达式

    • ^ 表示必须以 xxx 开头
    • $ 表示 必须以 xxx 结尾
